| Distribuce | Příkaz |
|------------|--------|
| Ubuntu/Debian | `sudo apt update` – aktualizace seznamu balíčků |
|  | `sudo apt upgrade` – aktualizace balíčků |
|  | `sudo apt install <balíček>` – instalace balíčku |
| Fedora | `sudo dnf install <balíček>` |
| CentOS/RHEL | `sudo yum install <balíček>` |
| Arch Linux | `sudo pacman -S <balíček>` |
