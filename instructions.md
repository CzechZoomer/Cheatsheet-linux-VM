##  Download a instalace VMware Workstation

1. Stáhnout instalační balíček:
   - Windows: [VMware Workstation Player](https://www.vmware.com/products/workstation-player.html)
   - Linux: dostupné `.bundle` balíčky na oficiálních stránkách.
2. Spustit instalátor:
   - Windows: dvojklik `.exe`, následovat průvodce
   - Linux:
     ```bash
     chmod +x VMware-Workstation-Full-*.bundle
     sudo ./VMware-Workstation-Full-*.bundle
     ```
3. Dokončit instalaci a spustit VMware Workstation.
