##  Instalace Linux OS ve VMware Workstation

1. **Nový virtuální stroj → Typical → Installer disk image (ISO) → Vybrat ISO**
2. Zvolit OS typ (Linux) a verzi (např. Ubuntu 64-bit)
3. Nastavit jméno VM a umístění souborů
4. Nastavit velikost disku (doporučeno ≥20 GB) a typ uložení (split disk / single file)
5. Dokončit průvodce a spustit VM
6. Projít instalací Linuxu (jazyk, uživatel, heslo, časové pásmo, rozdělení disku)
